# Staj Bulma Uygulaması - Basit Teknoloji Yığını

## 1. Önerilen Teknolojiler ve Nedenleri

| Katman | Teknoloji | Neden Seçiyoruz |
|--------|-----------|----------------|
| Frontend | **HTML + CSS + JavaScript (Vanilla)** | Başlangıç için en basit ve öğrenmesi kolay. Karmaşık framework’lere gerek yok. |
| Backend | **Python + Flask** | Flask, minimal ve öğrenmesi kolaydır. API oluşturmak ve AI ile entegre etmek kolaydır. |
| Yapay Zeka API | **Gemini API + Google AI Studio API Key** | Gemini API staj ilanlarını analiz ve eşleştirme için kullanılacak. Google AI Studio API Key, AI model entegrasyonu için hazır. |
| Veritabanı | **SQLite** | Başlangıç seviyesinde kurulumu ve kullanımı çok kolay, dosya tabanlı küçük projeler için ideal. |

---

## 2. Kurulum Adımları

### 2.1 Python Kurulumu

2. Kurulum sırasında **"Add Python to PATH"** seçeneğini işaretle
